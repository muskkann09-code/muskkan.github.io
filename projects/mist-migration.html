Project Overview

MIST helps teachers and administrators manage student records, attendance tracking, and report generation through an intuitive mobile interface. Built for educational institutions with varying levels of internet connectivity.

Key Features

  • Student profile creation, editing, and management
  • Attendance marking with bulk operations and quick actions
  • Attendance tracking with visual calendars and statistics
  • Report generation in multiple formats (PDF, Excel, CSV)
  • Offline data capture with automatic sync when online
  • Simple, intuitive UI designed for non-technical users
  • Class and section management for organizing students
  • Data export and sharing capabilities
  • Admin dashboard for overview and analytics

Technical Implementation

UI & Navigation: Implemented complete UI including student lists, attendance screens, and reports using FlutterFlow with custom data visualization.

Attendance System: Built Custom Widgets for attendance marking with bulk operations, calendar views, and statistics display.

Reporting Engine: Wrote Custom Functions for attendance calculations, report aggregation, and export logic with multiple format support.

Offline Architecture: Integrated local database for offline attendance capture with intelligent conflict resolution and background sync.

Cloud Integration: Implemented Firebase Authentication and Cloud Firestore for persistent records with real-time updates.

Tech Stack Used

FlutterFlow Dart Custom Functions Local DB (Offline Storage) Cloud Firestore Firebase Auth FCM Reporting Engine Play Console Excel/PDF Export

Challenges & Solutions

Challenge: Handling attendance for large numbers of students with quick, efficient UI.

Solution: Implemented virtualized lists, bulk operations, and keyboard shortcuts for rapid attendance marking.

Challenge: Generating complex reports with aggregated data across multiple time periods.

Solution: Created a reporting engine with pre-aggregated data and cached results for frequently accessed reports.

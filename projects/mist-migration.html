Digital Gram

Document management app with category & year-wise organization and instant search.

Key Features

  • Category-based navigation
  • Year-wise document filtering
  • Fast in-app text search inside documents

Technical Implementation

Implemented UI with FlutterFlow, created custom indexing functions for text search, integrated Firestore/Storage or REST sources for document metadata, and built a local caching layer for offline speed.

E-School Documents

PDF viewer app with text search, category & year organization, offline caching and Play Store deployment.

Key Features

  • Category & year-wise document listing
  • In-app PDF viewer with full-text search (custom index)
  • Offline caching using local DB
  • Firebase Storage for document hosting and Firestore metadata
  • FCM notifications for content updates

Technical Implementation

UI & navigation built in FlutterFlow. Custom Dart functions parse PDF text to build a searchable index. Firestore stores metadata; Storage holds files. Local DB (cache) ensures offline access. Integrated third-party PDF viewer and optimized rendering for low-end devices.

My Role

End-to-end: UI, custom functions, custom widgets, API & Firebase integration, local DB offline logic, FCM, performance tuning, build (AAB) & Play Store deployment, post-release monitoring & fixes.

Tech Stack

FlutterFlow · Dart (Custom functions) · Firebase Storage · Cloud Firestore · Local DB · FCM · PDF Viewer

Seva Book

Digital teacher service book for viewing & managing Seva Book entries with offline support.

Key Features

  • Display teacher/service records in a clean UI
  • Local DB to store records for offline viewing
  • Sync with Firestore when online
  • Push notifications for announcements (FCM)
  • Low-permission, privacy-first design

Technical Implementation

Created reusable custom widgets for record cards in FlutterFlow, custom functions to aggregate and filter history, and offline-first sync logic using a local DB. Integrated Firebase for optional remote sync and used FCM for announcements.

My Role

UI & UX design, custom widgets, custom functions, local DB logic & sync, Firebase integration, testing, performance tuning and Play Store deployment.

Tech Stack

FlutterFlow · Firestore · Local DB · FCM · Dart Custom Functions
